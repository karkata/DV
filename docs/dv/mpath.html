DV.mpath()

문법

let ret = DV.mpath(props);

파라미터

props 객체의 속성은 다음과 같습니다.

이름 형식 설명
href <iri>

이동 경로를 정의하는 'path' 요소를 가리키는 IRI 참조입니다.

반환

DVE 객체를 반환합니다.

설명

mpath 메소드는 animateMotion 요소의 서브 요소인 mpath 요소를 정의할 때 사용합니다.

<svg width="500" height="150">
    <path id="S01P01" d="M50,50 c50,-25 50,-25 100,0 s50,25 100,0 s50,-25 100,0 s50,25 100,0" stroke-width="1.5" stroke="gray" fill="none"/>
    <rect x="-5" y="-5" width="10" height="10" fill="red">
        <animateMotion begin="0s" dur="5s" rotate="auto" fill="freeze">
            <mpath xlink:href="#S01P01"/>
        </animateMotion>
    </rect>
</svg>

let svg = DV.svg().size(500,150).into("dv-01");
DV.path().mto(50,50).csto([
        [[50,-25],[50,-25],[100,0]],
        [[50,25],[100,0]],
        [[50,-25],[100,0]],
        [[50,25],[100,0]]
    ],true).nofill().stroke("gray",1.5).id("D01P01").into(svg);
DV.rect().dim(-5,-5,10,10).fill("red").into(svg).append(
    DV.animateMotion().time("0s","5s").motionRotate("auto").freeze().append(
        DV.mpath().xhref("#D01P01")
    )
);
