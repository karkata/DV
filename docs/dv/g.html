DV.prototype.g()

문법

var ret = DV.g();

파라미터

반환

dve 객체를 반환합니다.

설명

'g' 메소드는 'g' 요소를 생성합니다.

'g' 요소는 그룹처리할 때 사용하는 컨테이터 요소입니다.

'g' 요소로 그룹처리가 되면 id 속성을 사용하여 이름을 설정할 수 있으며, 이름이 주어진 그룹들은 애니메이션이나 재사용 되는 객체로 활용될 수 있습니다.

'g' 요소는 중첩이 가능합니다.

<svg width="200px" height="100px">
  <g id="grp1">
    <rect x="15" y="15" width="170" height="30" rx="15" fill="red"/>
  </g>
  <g id="grp2" transform="translate(0, 50)">
    <rect x="15" y="15" width="170" height="30" rx="15" fill="red"/>
    <g transform="translate(30, 30)">
      <circle cx="0" cy="0" r="10" fill="blue"/>
    </g>
  </g>
</svg>

var svg = DV.svg().size(200,100).into("dv-01");
var g1 = DV.g().into(svg);
DV.rect().dim(15,15,170,30).round(15).fill("red").into(g1);
var g2 = DV.g().translate(0,50).into(svg);
DV.rect().dim(15,15,170,30).round(15).fill("red").into(g2);
var g3 = DV.g().translate(30,30).into(g2);
DV.circle().center(0,0).radius(10).fill("blue").into(g3);
