DV.polyline()

문법

let ret = DV.polyline(props);

파라미터

'props' 파라미터 객체의 속성은 다음과 같습니다.

이름 형식 설명
points <list-of-points>|Array of point array

polyline을 생성하는 점의 좌표들입니다. 좌표는 공백문자 혹은 콤마(,)로 구분을 합니다.

반환

DVE 객체를 반환합니다.

설명

'polyline' 메소드는 'polyline' 요소를 생성합니다.

연결된 직선의 선분(세그먼트) 집합을 그립니다. polyline은 열린 도형으로 정의합니다.

<svg width="400px" height="150px">
  <polyline stroke="blue" stroke-width="5" points="50,125 100,125 100,100 150,100 150,75 200,75 200,50 250,50 250,25 300,25"/>
  <polyline stroke="blue" stroke-width="5" points="125,125 175,125 175,100 225,100 225,75 275,75 275,50 325,50 325,25 375,25" fill="none"/>
</svg>

let svg = DV.svg().size(400,150).into("dv-01");
DV.polyline({ "points": "50,125 100,125 100,100 150,100 150,75 200,75 200,50 250,50 250,25 300,25" }).stroke("blue",5).into(svg);
DV.polyline().points([
    [125,125],
    [175,125],
    [175,100],
    [225,100],
    [225,75],
    [275,75],
    [275,50],
    [325,50],
    [325,25],
    [375,25]
]).stroke("blue",5).fill("none").into(svg);
