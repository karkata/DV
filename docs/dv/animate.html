DV.animate()

문법

let ret = DV.animate(props);

파라미터

반환

DVE 객체를 반환합니다.

설명

animate 메소드는 animate 요소를 생성합니다.

animate 요소는 어떤 속성을 시간의 흐름에 따라 변화시키는 애니메이션하고자 할 때 사용합니다.

예를 들어, 상자(rect)의 불투명도(opacity)를 시간에 따라 변환시킬 수 있습니다.

<svg width="200" height="100">
    <rect x="75" y="25" width="50" height="50" fill="red">
        <animate attributeName="opacity" attributeType="CSS" from="1" to="0" begin="0s" dur="3s" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 애니메이션을 수행할 수 있습니다.

let svg = DV.svg().size(200,100).into("dv-01");
let rect = DV.rect().dim(75,25,50,50).fill("red").into(svg);
DV.animate().toWhat("opacity").fromToBy(1,0).time("0s","3s").freeze().into(rect);

'href' 속성을 사용하면 애니메이션의 대상이 되는 요소를 참조 형식으로 설정할 수 있습니다.

<svg width="200" height="100">
    <rect x="75" y="25" width="50" height="50" fill="red" id="S02R1"/>
    <animate xlink:href="#S02R1" attributeName="opacity" attributeType="CSS" from="1" to="0" begin="0s" dur="3s" fill="freeze"/>
</svg>

DV를 사용하면 다음과 같습니다.

let svg = DV.svg().size(200,100).into("dv-02");
DV.rect().dim(75,25,50,50).fill("red").id("D02R01").into(svg);
DV.animate().toWhat("opacity").xhref("#D02R01").fromToBy(1,0).time("0s","3s").freeze().into(svg);
