DV.styleblock()

문법

let ret = DV.styleblock(css);

파라미터

css String

CSS 스타일 정의입니다.

반환

DVE 객체를 반환합니다.

설명

'styleblock' 메소드는 'style' 요소를 생성합니다.

'style' 요소는 SVG 문서 내부에 스타일 시트를 포함시켜 사용할 때 사용됩니다.

<svg width="400px" height="100px">
    <style type="text/css">
        #S01R01.common-border-style { stroke: red; stroke-width: 5px; }
    </style>
    <rect id="S01R01" x="10" y="10" width="100" height="50" class="common-border-style" fill="orange"/>
</svg>

let svg = DV.svg().size(400,100).into("dv-01");
DV.styleblock().txt("#D01R01.common-border-style { stroke: red; stroke-width: 5px; }").into(svg);
DV.rect().dim(10,10,100,50).fill("orange").id("D01R01").class("common-border-style").into(svg);
