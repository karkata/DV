DV.rect()

문법

let ret = DV.rect([props]);

파라미터

'props' 파라미터는'rect' 요소를 생성하는데 필요한 속성을 갖는 객체입니다.

이름 형식 설명
x, y <coordinate>

렌더링할 엘리먼트가 놓일 사각형의 영역입니다. 기본값은 0입니다.
width, height <length>

width, height는 엘리먼트의 크기이며, 0이면 렌더링을 하지 않고, 음수값이 설정되면 오류가 발생합니다.
rx, ry <length>

사각형의 각 코너 끝을 타원형으로 만들기 위한 x축, y축 반지름 값입니다. 음수값이 설정되면 오류가 발생합니다.

반환

dve 객체를 반환합니다.

설명

'rect' 메소드는 'rect' 엘리먼트를 생성합니다.

'rect' 엘리먼트를 사용하여 사각형 모양을 그릴 수 있습니다.

<svg width="400px" height="150px">
    <rect x="25" y="25" width="50" height="50"/>
    <rect x="100" y="25" width="50" height="50" fill="red"/>
    <rect x="175" y="25" width="50" height="50" fill="red" stroke="purple" stroke-width="3"/>i
    <rect x="250" y="25" width="50" height="50" fill="red" stroke="purple" stroke-width="3" rx="5" ry="5"/>
</svg>

let svg = DV.svg().size(400,150).into("dorothy-01");
DV.rect().dim(25,25,50,50).into(svg);
DV.rect().dim(100,25,50,50).fill("red").into(svg);
DV.rect().dim(175,25,50,50).fill("red").stroke("purple",3).into(svg);
DV.rect().dim(250,25,50,50).round(5,5).fill("red").stroke("purple",3).into(svg);
