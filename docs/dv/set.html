DV.set()

문법

let ret = DV.set();

파라미터

반환

DVE 객체를 반환합니다.

설명

set 메소드는 set 요소를 생성합니다.

set 요소는 주어진 지속 시간동안 애니메이션 대상의 특정 속성을 변경하는 단순한 방법을 제공합니다.

set 요소는 문자열 및 부울 값과 같이 합리적으로 보간할 수 없는 속성 유형을 포함하여 모든 속성 유형을 지원합니다.

set 요소는 비-가산적입니다. additive, accumulate 속성들이 허용되지 않으며, 지정된 경우 무시됩니다.

<svg width="200" height="100">
    <rect x="75" y="25" width="50" height="50" fill="red">
        <!-- 3초 후 가로 크기가 100으로 증가한 후 2초가 지나면 원상복귀 합니다. -->
        <set attributeName="width" attributeType="XML" to="100" begin="3s" dur="2s"/>
        <set attributeName="x" attributeType="XML" to="50" begin="3s" dur="2s"/>
    </rect>
</svg>

let svg = DV.svg().size(200,100).into("dv-01");
let rect = DV.rect().dim(75,25,50,50).fill("red").into(svg);
DV.set().toWhat("width").keyTo("100").time("3s","2s").into(rect);
DV.set().toWhat("x").keyTo("50").time("3s","2s").into(rect);
