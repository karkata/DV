DV.prototype.getFuncIRI()

문법

let ret = DV.getFuncIRI(value);

파라미터

이름 형식 설명
value String|DVE

객체의 ID 문자열이거나 객체 그 자체입니다.

반환

객체의 FuncIRI 값을 반환합니다. FuncIRI 값은 'url(#ID)' 형식의 문자열을 나타냅니다.

설명

getFuncIRI은 주어진 객체의 ID를 FuncIRI 형식에 맞도록 변경하여 반환하거나 파라미터가 객체인 경우에는 그 객체의 ID 값을 가지고 FuncIRI 값을 만들어 반환합니다.

<svg width="300px" height="150px">
    <defs>
        <linearGradient id="S01GD01">
            <stop offset="10%" stop-color="#5d4037"/>
            <stop offset="90%" stop-color="#d7ccc8"/>
        </linearGradient>
    </defs>
    <rect x="0" y="25" width="100" height="100" fill="url(#S01GD01)"/>
</svg>

let svg = DV.svg().size(300,150).into("svg-01");
let def = DV.defs().into(svg);
let grad1 = DV.gradl().id("D01GD01").into(def);
DV.stop().oco(0.1,"#5d4037").into(grad1);
DV.stop().oco(0.9,"#d7ccc8").into(grad1);
DV.rect().dim(0,25,100,100).fill(DV.getFuncIRI(grad1)).into(svg);
