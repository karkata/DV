DV.line()

문법

let ret = DV.line(props);

파라미터

props 파라미터의 속성은 다음과 같습니다.

이름 형식 설명
x1, y1, x2, y2 <coordinate>

시작점(x1, y1), 끝점(x2, y2)를 나타냅니다.

반환

DVE 객체를 반환합니다.

설명

line 메소드는 line 요소를 생성합니다.

line 요소는 시작점과 끝점을 사용하여 직선의 선분(세그먼트)을 그립니다.

<svg width="300px" height="150px">
  <line x1="10" y1="10" x2="50" y2="100" stroke="green"/>
  <line x1="50" y1="10" x2="100" y2="100" stroke="green" stroke-width="5"/>
  <line x1="100" y1="10" x2="150" y2="100" stroke="green" stroke-width="10"/>
  <line x1="150" y1="10" x2="200" y2="100" stroke="green" stroke-width="15" stroke-dasharray="5 5"/>
</svg>

let svg = DV.svg().size(300,150).into("dv-01");
DV.line().point1(10,10).point2(50,100).stroke("green").into(svg);
DV.line().bound(50,10,100,100).stroke("green",5).into(svg);
DV.line().bound(100,10,150,100).stroke("green",10).into(svg);
DV.line().bound(150,10,200,100).stroke("green",15).strokeDash("5 5").into(svg);
