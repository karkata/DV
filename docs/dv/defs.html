DV.defs()

문법

let ret = DV.defs();

파라미터

반환

dve 객체를 반환합니다.

설명

'defs' 메소드는 'defs' 요소를 생성합니다.

'defs'는 참조되어지는 요소들을 담는 컨테이너 요소입니다.

'defs' 하위에 정의된 요소들은 바로 렌더링되지 않으나, 소스 트리상에 실질적으로 존재하는 객체입니다. 그러므로 다른 요소들에 의해 참조가 가능한 것입니다.

<svg width="200px" height="100px">
  <defs>
    <linearGradient id="S01LG01">
      <stop offset="20%" stop-color="#39F"/>
      <stop offset="90%" stop-color="#F3F"/>
    </linearGradient>
  </defs>
  <rect x="50" y="25" width="100" height="50" fill="url(#S01LG01)"/>
</svg>

let svg = DV.svg().size(200,100).into("dv-01");
let defs = DV.defs().into(svg);
let grd = DV.gradl().id("D01LG01").into(defs);
DV.stop().stopOCO("20%","#39F").into(grd);
DV.stop().stopOCO("90%","#F3F").into(grd);
DV.rect().dim(50,25,100,50).fill(grd).into(svg);
