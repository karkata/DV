DV.link()

문법

let ret = DV.link(props);

파라미터

'props' 파라미터는 'a' 요소를 생성하는데 필요한 속성을 갖는 객체입니다.

이름 형식 설명
href <IRI>

참조되는 대상의 위치를 나타내는 <IRI> 형식의 값
target string

연결 링크가 보여질 대상을 나타내는 문자열

반환

dve 객체를 반환합니다.

설명

'link' 메소드는 'a' 요소를 생성합니다.

'a'는 하이퍼링크를 제공하는 SVG anchor 요소입니다.

<svg width="200px" height="100px">
    <a xlink:href="https://www.w3.org/TR/SVG11">
        <rect x="15" y="35" width="170" height="30" rx="15" fill="red"/>
        <text text-anchor="middle" x="100" y="55" fill="blue">SVG 1.1</text>
    </a>
</svg>

let svg = DV.svg().size(200,100).into("svg-01");
let a = DV.link().xhref("https://www.w3.org/TR/SVG11").into(svg);
let r = DV.rect().dim(15,35,170,30).round(15).fill("red").into(a);
DV.text().xys(100,55).text("SVG 1.1").tanchor("middle").fill("white").into(a);
