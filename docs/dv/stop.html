DV.stop()

문법

let ret = DV.stop(props);

파라미터

'props' 파라미터의 속성은 다음과 같습니다.

이름 형식 설명
offset <number>|<percentage>

'stop' 요소가 놓이는 위치입니다.

0부터 1사이의 값이거나, 퍼센티지(보통 0~100%) 값입니다.
stop-color <color>|<icccolor>

'stop' 요소가 놓이는 위치의 색입니다.
stop-opacity <opacity-value>

'stop' 요소가 놓이는 위치의 불투명도를 나타냅니다.

반환

DVE 객체입니다.

설명

gradients 효과를 표현할 때 주어진 벡터 방향에서 색상이 변화되는 위치를 'stop' 요소로 표현합니다.

'stop' 메소드는 현재 요소가 'linearGradient, radialGradient'일 때, 'stop' 요소를 추가하는 작업을 수행합니다.

<svg width="300px" height="150px">
    <defs>
        <linearGradient id="S01GD01">
            <stop offset="10%" stop-color="#5d4037"/>
            <stop offset="90%" stop-color="#d7ccc8"/>
        </linearGradient>
        <radialGradient id="S01GD02" cx="50%" cy="50%" r="50%">
            <stop offset="10%" stop-color="#5d4037"/>
            <stop offset="90%" stop-color="#d7ccc8"/>
        </linearGradient>
    </defs>
    <rect x="0" y="25" width="100" height="100" fill="url(#S01GD01)"/>
    <rect x="150" y="25" width="100" height="100" fill="url(#S01GD02)"/>
</svg>

let svg = DV.svg().size(300,150).into("dv-01");
let def = DV.defs().into(svg);
let grad1 = DV.gradl().id("D01GD01").into(def);
DV.stop().oco(0.1,"#5d4037").into(grad1);
DV.stop().oco(0.9,"#d7ccc8").into(grad1);
let grad2 = DV.gradr().id("D01GD02").into(def);
DV.stop().oco(0.1,"#5d4037").into(grad2);
DV.stop().oco(0.9,"#d7ccc8").into(grad2);
DV.rect().dim(0,25,100,100).fill(grad1).into(svg);
DV.rect().dim(150,25,100,100).fill(grad2).into(svg);
