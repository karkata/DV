DV

DV는 SVG 요소들을 생성하는 정적 메소드들을 가지고 있는 객체이다.

DV 객체는 dv.shape.js 파일을 참조하면 생성되는 글로벌 객체인데, dv.core.js 파일을 참조하면 사용할 수 있는 DVC 객체만으로 가지고 SVG 요소를 생성하는 과정은 약간 복잡하기 때문에, 간단한 노력으로 SVG 요소를 생성할 수 있도록 도움을 주는 정적 메소드들을 DV 객체를 통해서 제공받을 수 있다.
