DV.desc()

문법

let ret = DV.desc();

파라미터

반환

dve 객체를 반환합니다.

설명

'desc' 메소드는 'desc' 요소를 생성합니다.

'desc' 요소는 설명을 하기 위한 용도로 사용되며, 화면에 렌더링되지 않습니다.

<svg width="200px" height="100px">
    <g style="cursor:pointer">
        <desc>Nothing will happen.</desc>
        <rect x="15" y="15" width="170" height="30" rx="15" fill="red"/>
        <text x="100" y="35" text-anchor="middle" fill="white">Click me!</text>
    </g>
</svg>

let svg = DV.svg().size(200,100).into("dv-01");
let g = DV.g().css("cursor", "pointer").into(svg);
DV.desc().txt("Nothing will happen.").into(g);
DV.rect().dim(15,15,170,30).round(15).fill("red").into(g);
DV.text().txt("No action").xys(100,35).fill("white").attr("text-anchor", "middle").into(g);
