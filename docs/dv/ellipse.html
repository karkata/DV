DV.ellipse()

문법

let ret = DV.ellipse(props);

파라미터

'props' 파라미터는 'ellipse' 요소를 생성하는데 필요한 속성을 갖는 객체입니다.

이름 형식 설명
cx, cy <coordinate>

원의 중점을 나타내는 위치입니다.

설정을 하지 않으면 (0, 0) 위치를 가리킵니다.
rx, ry <coordinate>

타원의 두 반지름을 나타냅니다.

rx는 x 축 방향의 반지름, ry는 y 축 방향의 반지름을 나타냅니다.

0이면 렌더링을 하지 않으며, 음수값이 설정되면 오류가 발생합니다.

반환

dve 객체를 반환합니다.

설명

'ellipse' 메소드는 'ellipse' 엘리먼트를 생성합니다.

<svg width="250px" height="150px">
  <ellipse cx="125" cy="75" rx="100" ry="50" stroke="blue" stroke-width="10" fill="red"/>
</svg>

let svg = DV.svg().size(250,150).into("dv-01");
DV.ellipse().center(125,75).round(100,50).fill("red").stroke("blue", 10).into(svg);
