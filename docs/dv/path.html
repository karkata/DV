DV.path()

문법

let ret = DV.path(props);

파라미터

'props' 파라미터의 속성은 다음과 같습니다.

이름 형식 설명
d String

패스를 표현하는 좌표 데이터입니다.
pathLength <length>

패스의 전체 길이에 대한 계산값입니다.

이 값은 작성자가 직접 계산한 사용자 단위의 값입니다. 이 값은 에이전트 자신의 거리 경로 계산을 교정자의 교정과 함께 보정하는데 사용됩니다.

반환

DVE 객체를 반환합니다.

설명

'path' 메소드는 SVG 'path' 요소를 생성합니다.

패스는 도형의 외곽선(outline)을 나타내며, 점과 점 사이를 직선 또는 곡선으로 연결하여 표현합니다.

SVG에서 패스는 'path' 요소를 사용하며, 'd' 속성을 사용하여 패스를 표현하는 좌표 데이터를 기록합니다.

<svg width="5cm" height="4cm" viewBox="0 0 500 400">
  <path fill="none" stroke="blue" stroke-width="5" d="M100,100 L400,100 L400,300 L100,300 v-100 h100 v-50 h-100 z"/>
</svg>

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-01");
DV.path().d("M100,100 L400,100 L400,300 L100,300 v-100 h100 v-50 h-100 z").fill("none").stroke("blue", 5).into(svg);

'd' 메소드를 사용하여 패스를 그릴 수도 있지만, 다른 메소드를 함께 사용하여 패스를 그리를 것을 권장합니다.

패스를 그리는 메소드들: 'mto, lto, vto, hto, cto, csto, qto, qsto, arc,arcf, pz
