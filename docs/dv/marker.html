DV.prototype.marker()

문법

var ret = DV.marker(props);

파라미터

'props' 파라미터의 속성은 다음과 같습니다.

이름 형식 설명
refX, refY <coordinate>

마커가 테두리 위의 점에 올려질 위치로써, viewBox의 좌표계를 기준으로 합니다.

만일 테두리 위의 점의 가운데에 마커가 오도록 하려면, width(markerWidth), height(markerHeight)의 반에 해당하는 값으로 설정해야 합니다.

기본값은 0입니다.
markerWidth, markerHeight <length>

markerWidth, markerHeight는 마커의 크기입니다. 기본값은 3입니다.

preserveAscpectRatio 속성에 따라 viewBox의 가로/세로 비율로 표현되기 때문에 반드시 markerWidth, markerHeight 크기로 보이지 않을 수 있습니다.
orient "auto"|<angle>

auto 또는 회전 각도를 설정합니다. orient 속성은 그려지는 마커를 회전시킬 때 사용됩니다.

auto이면 테두리의 방향(접선)의 동일한 위치가 됩니다.
markerUnits "strokeWidth"|"userSpaceOnUse"

'strokeWidth'이면 마커는 테두리의 두께에 따라 스케일링이 발생합니다.

'useSpaceOnUse'이면 테두리의 두께와 상관없이 markerWidth, markerHeight 만큼 마커가 그려집니다.

반환

dve 객체를 반환합니다.

설명

'marker' 메소드는 'marker' 요소를 생성합니다.

마커는 'path', 'line', 'polyline', 'polygon' 요소가 갖는 점(또는 점 마다)에 붙는 모양을 의미합니다.

일반적으로 양 끝에서 화살표 머리를 만들거나 점마다 동일한 모양의 붙혀야 하는 경우 사용됩니다.

사용될 모양은 'marker' 요소로 만들고, 각 요소가 갖는 'marker-start, marker-mid, marker-end' 속성을 사용하여 요소에 모양을 지정하게 됩니다.

'marker' 요소 내부에 그려지는 요소는 'marker' 요소의 'viewBox' 영역 내에 그려진다고 보면 됩니다.

'markerWidth, markerHeight'는 마커의 크기(viewBox가 그려질)를 말하며, 마커를 사용하는 테두리의 두께가 1인 경우를 기준으로 scaling됩니다.

만일 scaling 없이 사용하려면 'markerUnits' 속성을 'strokeWidth'가 아닌 'useSpaceOnUse'를 사용하여 스케일링없이 선언된 그대로 그려지게 할 수 있습니다.

'refX, refY'는 마커가 테두리위의 점에 올려질 위치로써, 'viewBox'의 좌측/상단은 0, 0입니다.

'orient' 속성은 그려지는 마커의 회전각을 설정할 때 사용합니다.

<svg width="300" height="100">
    <defs>
        <marker id="start" refX="20" refY="5" markerWidth="10" markerHeight="10" viewBox="0 0 20 10" preserveAspectRatio="none">
            <ellipse cx="10" cy="5" rx="9" ry="4" fill="none" stroke="red" stroke-width="1"/>
        </marker>
        <marker id="middle" refX="10" refY="10" markerWidth="10" markerHeight="10" viewBox="0 0 20 10" orient="90">
            <ellipse cx="10" cy="5" rx="9" ry="4" fill="none" stroke="Red" stroke-width="1"/>
        </marker>
        <marker id="end" refX="0" refY="5" markerWidth="10" markerHeight="10" viewBox="0 0 10 10" orient="auto" >
            <path d="M1,1 L9,5 L1,9 z" fill="none" stroke="red" stroke-width="1"/>
        </marker>
    </defs>
    <path d="M50,25 L100,25 L150,50" fill="none" stroke="black" stroke-width="3"
        marker-start="url(#start)" marker-mid="url(#middle)" marker-end="url(#end)"/>
</svg>

var svg = DV.svg().size(300,100).into("dv-01");
DV.defs().appD01E01(
    DV.marker().markerPoint(20,5).markerSize(10,10).viewbox(0,0,20,10).aspectRatio("none").id("D01S01").append(
        DV.ellipse().center(10,5).round(9,4).fill("none").stroke("red", 1)
    ),
    DV.marker().markerPoint(10,10).markerSize(10,10).markerOrient(90).viewbox(0,0,20,10).id("D01M01").append(
        DV.ellipse().center(10,5).round(9,4).fill("none").stroke("red", 1)
    ),
    DV.marker().markerDim(0,5,10,10,"auto").viewbox(0,0,10,10).id("D01E01").append(
        DV.path().d("M1,1 L9,5 L1,9 z").fill("none").stroke("red", 1)
    )
).into(svg);
DV.path().d("M50,25 L100,25 L150,50").fill("none").stroke("black", 3).mark("#D01S01","s").mark("#D01M01","m").mark("#D01E01","e").into(svg);
