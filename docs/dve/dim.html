DVE.prototype.dim()

문법

let ret = object.dim(x[, y[, width[, height]]]);

파라미터

이름 형식 설명
x, y <coordinate>

SVG 요소 x축, y축 위치를 설정합니다.
width, height <length>

SVG 요소의 가로, 세로 길이를 설정합니다.

반환

DVE 객체를 반환합니다.

설명

'dim' 메소드는 SVG 요소 바운딩 영역을 설정합니다.

'dim' 메소드는 user space 영역에 SVG 요소가 렌더링되는 위치를 변경할 때 사용합니다.

let svg = new dv().svg(200, 100).into("dv-01");
let rect = svg.rect(0, 0, 100, 20).fill("red");
// from {0, 0, 100, 20} to {50, 35, 100, 30}
rect.dim(50,35,100,30);
