DVE.prototype.attr()

문법

let ret = object.attr(name, value[, appending]);

파라미터

이름 형식 설명
name String

속성의 이름입니다.
value Boolean|null|undefined|Number|String|Object

속성의 값입니다.
appending Boolean

속성에 값을 설정할 때, 기존 속성 값 뒤에 추가하려면 true, 덮어쓰려면 false 입니다.

반환

attr 메소드의 파라미터가 이름만 주어진다면, 해당 이름으로 지정된 속성의 값이 반환됩니다.

이와 달리 값 파라미터가 주어진다면, DVE 객체가 반환됩니다.

설명

'attr' 메소드는 요소가 갖는 속성값을 반환하거나, 속성값을 설정할 때 사용합니다.

name 파라미터만 주어지면 현재 대상 속성값이 반환되고, value 파라미터까지 주어진다면 대상 속성값을 주어진 value로 설정합니다.

속성값이 반환될 때와 달리, 속성값을 설정할 때에는 속성을 가지고 있는 요소의 dv 래퍼 객체가 반환됩니다.

let svg = new dv().svg(400, 70).into("dv-01");
let rect1 = svg.rect(10, 10, 200, 20).into(svg).attr("fill", "blue");
// rect1의 fill 속성값을 두 번째 사각형의 fill 속성값으로 설정합니다.
svg.rect(10, 40, 200, 20).fill(rect1.attr("fill")).into(svg);
