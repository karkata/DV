DVE.prototype.focalPoint()

문법

let ret = object.focalPoint(fx[, fy]);

파라미터

이름 형식 이름
fx, fy <coordinate>

radial gradients의 초점 위치를 나타내는 (fx, fy)입니다.

fx, fy 파라미터가 undefined 이면, 'radialGradient'의 cx, cy 값과 일치하게 됩니다.

fx, fy는 focal point를 나타냅니다. cx, cy, r이 gradients가 렌더링되는 영역을 나타낸다면, focal point는 gradients 중심을 의미합니다.

반환

DVE 객체를 반환합니다.

설명

radialGradient 요소를 사용해서 radial(방사형) gradients를 표현할 때, 촛점(focal point)를 정의합니다.

gradients 효과가 시작되고 끝나는 vector는 cx, cy, r로 결정이 되는데, cx, cy, r이 gradients가 렌더링되는 영역을 나타낸다면, focal point는 gradients 시작되는 중심을 의미합니다.

let svg = DV.svg().size(600,250).into("dv-01");
let def = DV.defs().into(svg);
let grad1 = DV.gradr().center("25%","25%").radius(0.5).id("D01RG01").into(def);
DV.stop().oco("0%","red").into(grad1);
DV.stop().oco("100%","blue").into(grad1);

let grad2 = DV.gradr().center("25%","25%").radius(0.5).focalPoint("50%","50%").id("D01RG02").into(def);
DV.stop().oco("0%","red").into(grad2);
DV.stop().oco("100%","blue").into(grad2);

let g1 = DV.g().fsize("0.80em").translate(50,50).into(svg);
DV.text().txt("cx = 25%, cy = 25%, r = 0.5").xys(0,17).into(g1);
DV.rect().dim(0,25,200,100).fill(grad1).into(g1);
let svg1 = DV.svg().dim(0,25,200,100).into(g1);
DV.ellipse().center("25%","25%").round("50%","50%").stroke("white",1).nofill().into(svg1);
DV.circle().center("25%","25%").radius("1%").fill("white").into(svg1);
let g2 = DV.g().fsize("0.80em").translate(300,50).into(svg);
DV.text().txt("cx = 25%, cy = 25%, r = 0.5, fx = 50%, fy = 50%").xys(0,17).into(g2);
DV.rect().dim(0,25,200,100).fill(grad2).into(g2);
let svg2 = DV.svg().dim(0,25,200,100).into(g2);
DV.ellipse().center("25%","25%").round("50%","50%").stroke("white",1).nofill().into(svg2);
DV.circle().center("50%","50%").radius("1%").fill("white").into(svg2);
