DVE.prototype.into()

문법

var ret = dve.into(container);

파라미터

이름 형식 설명
container String|DOM object|DVE

현재 DVE 객체의 부모가 될 객체입니다.

반환

DVE 객체를 반환합니다.

설명

'into' 메소드는 SVG 요소를 감싼 현재 객체를 파라미터로 주어진 객체의 자식 객체로 등록합니다.

DVE 객체를 자식 또는 부모 객체로 등록하면 SVG 요소들도 동일하게 부모, 자식 구조를 빌딩합니다. 

var svg = DV.svg().size(400,100).into("dv-01");
// svg 요소 하위에 rect 요소가 생성됩니다.
var r = DV.rect().dim(10,10,100,30).fill("#7c4dff").into(svg);
var g = DV.g().into(svg);
// rect 요소를 svg 요소에서 제거한 후, g 요소 하위로 이동시킵니다.
r.into(g);
