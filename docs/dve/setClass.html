DVE.prototype.class()

문법

var ret = dve.class([className[, append]]);

파라미터

이름 형식 설명
className String

CSS의 클래스 이름입니다.
append Boolean

기존 클래스 이름을 변경하지 않고, 추가하려면 true로 설정합니다.

반환

'class' 메소드에 파라미터 인자값이 주어지지 않았다면 해당 SVG 요소의 클래스 이름을 반환합니다. 만일 파리미터 인자값이 주어진다면, 해당 요소의 클래스 속성에 주어진 파라미터 값을 설정하고, DVE 객체를 반환합니다.

설명

'class' 메소드는 요소마다 가지고 있는 'class' 속성값을 가져오거나 설정할 때 사용합니다.

var svg = DV.svg().size(275,150).into("dv-01");

// 테두리 두께가 5이고 안을 채우지 않는 스타일을 클래스로 선언합니다.
DV.styleblock().txt(".circle-ui-default { stroke-width: 5; fill: none; }").into(svg);

var c1 = DV.circle().center(75,75).radius(50).class("circle-ui-default").stroke("red").into(svg);
// 같은 클래스를 갖도록 합니다.
DV.circle().center(200,75).radius(50).class(c1.class()).stroke("orange").into(svg);
