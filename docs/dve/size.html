DVE.prototype.size()

문법

let ret = object.size(width[, height]);

파라미터

이름 형식 설명
width <length>

요소의 가로 길이(크기) 값입니다.
height <length>

요소의 세로 길이(크기) 값입니다.

반환

DVE 객체를 반환합니다.

설명

'size' 메소드는 요소의 'width', 'height' 속성값을 설정합니다.

<svg width="300" height="60">
  <rect x="10" y="10" width="50" height="20" fill="red"/>
  <rect x="10" y="40" width="50" height="20" fill="blue"/>
</svg>

let svg = DV.svg().size(300,60).into("dv-01");
DV.rect().dim(10,10,50,20).fill("red").into(svg);
DV.rect().point(10,40).size(50,20).fill("blue").into(svg);
