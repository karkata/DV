DVE.prototype.rmattr()

문법

let ret = object.rmattr(name[, value]);

파라미터

이름 형식 설명
name String

속성 이름
value String

속성값

반환

DW(DV Wrapper) 객체입니다.

설명

rmattr 메소드는 요소가 갖는 속성을 삭제하거나, 속성 값만 삭제할 때 사용합니다.

name 파라미터만 주어지면 현재 대상 속성이 삭제되고, value 파라미터까지 주어진다면 대상 속성 값 중에서 value를 찾아 삭제합니다.

let svg = DV.svg().size(400,70).into("dv-01");
let rect = DV.rect().dim(10,10,200,20).fill("blue").stroke("red",5).into(svg);
rect.rmattr("stroke");
