DVE.prototype.keyValues()

문법

let ret = object.keyValues(values);

파라미터

이름 형식 설명
values <list>

세미콜론으로 구분된 하나 또는 그 이상의 애니메이션 값입니다.

값은 SVG에서만 사용되는 특별한 형식을 제외하고, 표준적으로 SMIL 스펙을 따릅니다.

반환

DVE 객체를 반환합니다.

설명

keyValues 메소드는 animate, animateMotion 요소의 values 속성을 설정합니다. values 속성은 애니메이션 대상(attributeName 속성으로 지정됩니다.)의 값을 의미하며, 세미콜론(;) 문자를 사용하여 목록으로도 설정할 수 있습니다.

주어진 파라미터는 배열 형식인 경우, 세미콜론 문자를 구분자로 갖는 단일 문자열로 변환한 값으로 대체하게 됩니다.

keyValues 메소드를 사용하게 되면, 각 애니메이션 값에 대응하는 시간 정보를 keyTimes 메소드를 사용하여 설정해줘야 합니다.

<svg width="400" height="20">
    <rect x="50" y="0" width="10" height="10" fill="red">
        <!--
            박스를 x 좌표 상에서 0부터 390까지 2초동안 이동시킵니다.
            0.5초 동안 오른쪽으로 200, 0.5초 동안 다시 왼쪽으로 100, 남은 1초간 다시 오른쪽으로 290만큼 이동합니다.
        --> 
        <animate attributeName="x" attributeType="XML" values="0;200;100;390" keyTimes="0;0.25;0.5;1" dur="2s" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 효과를 보여줄 수 있습니다.

let svg = DV.svg().size(400,20).into("dv-01");
DV.rect().dim(50,0,10,10).fill("red").append(
    DV.animate().toWhat("x").keyValues([0,200,100,390]).time("0s","2s").keyTimes([0,0.25,0.5,1]).freeze()
).into(svg);

KeyValues 메소드를 사용하면, keyFrom, keyTo, keyBy, fromToBy을 사용하여 설정된 속성이 무시됩니다.
