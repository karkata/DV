DVE.prototype.css()

문법

let ret = object.css(sname[, svalue]);

파라미터

이름 형식 설명
sname String

CSS 속성 이름
svalue String

css 속성 값

반환

svalue 파라미터가 주어지지 않았다면, SVG 요소가 갖는 sname 속성 값을 반환합니다. 만일 svalue 파라미터가 주어지면, sname이 가리키는 속성을 svalue 값으로 설정하고 DVE 객체를 반환합니다.

설명

css 메소드는 SVG 요소가 갖는 인라인 CSS(Cascading Style Sheets) 속성값을 반환하거나, 속성값을 설정할 때 사용합니다.

sname 파라미터만 주어지면 현재 대상 속성값이 반환되고, svalue 파라미터까지 주어진다면 대상 속성값을 주어진 값으로 설정합니다.

속성값이 반환될 때와 달리, 속성값을 설정할 때에는 속성을 가지고 있는 요소의 DVE 객체가 반환됩니다.

css 메소드는 HTML 요소에 인라인 스타일을 지정하는 것처러럼 사용합니다.

let svg = DV.svg().size(400,100).into("dv-01");
DV.rect().dim(10,10,200,20).css("fill","red").into(svg);
DV.rect().dim(10,40,200,20).attr("fill","green").into(svg);
DV.rect().dim(10,70,200,20).fill("blue").into(svg);

SVG에서 사용되는 스타일은 CSS2와 동일한 것도 있지만, SVG에서만 독자적으로 사용되는 속성들이 많습니다.

SVG에서 사용되는 스타일은 링크에서 확인하시길 바랍니다.
