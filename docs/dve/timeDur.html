DVE.prototype.timeDur()

문법

var ret = object.timeDur(value);

파라미터

이름 형식 설명
value string, <Clock-value>

애니메이션의 단순 기간을 지정합니다.

dur 값은 <Clock-value>, 'media', 'indefinite'로 나타낼 수 있습니다.

  • Clock-value: 단순 지속 시간을 나타내는 시간의 길이 값으로써, SVG 문서 조각이 로드된 시점을 기준으로 측정된 타임라인의 위치를 의미합니다. 값은 0 보다 커야합니다.
  • 'media': 내장 미디어 기간으로 단순 지속 시간을 지정합니다. 미디어를 정의하는 요소에만 유효하며, SVG 애니메이션에서는 무시됩니다.
  • 'indefinite': 단순 지속 시간을 무기한으로 지정합니다.

반환

DVE 객체를 반환합니다.

설명

timeDur 메소드는 animate, set, animateMotion, animateTransform 요소의 dur 속성을 설정합니다.

SVG 애니메이션은 begin, dur, end 속성을 사용하여 애니메이션의 동작하는 시간을 설정할 수 있습니다.

<svg width="400" height="100">
    <rect x="50" y="0" width="10" height="10" fill="red">
        <!-- 5번 반복 -->
        <animate attributeName="x" attributeType="XML" from="50" to="390" begin="0s" dur="3s" fill="freeze" id="S01R01" repeatCount="5"/>
    </rect>
    <rect x="50" y="30" width="10" height="10" fill="red">
        <!-- SVG 문서 조각이 로드된 후 1초 지나 시작 -->
        <animate attributeName="x" attributeType="XML" from="50" to="390" begin="1s" dur="2s" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 효과를 보여줄 수 있습니다.

var svg = DV.svg().size(400,100).into("dv-01");
DV.rect().dim(50,0,10,10).fill("red").append(
    DV.animate().toWhat("x").fromToBy(50,390).timeBegin("0s").timeDur("3s").repeatCount(5).freeze().id("D01R01")
).into(svg);
DV.rect().dim(50,30,10,10).fill("red").append(
    DV.animate().toWhat("x").fromToBy(50,390).timeBegin("1s").timeDur("2s").freeze()
).into(svg);
