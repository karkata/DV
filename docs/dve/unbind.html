DVE.prototype.unbind()

문법

let ret = object.unbind(type, listener);

파라미터

이름 형식 설명
type String 삭제하려는 이벤트 형식을 나타내는 문자열
listener Function|EventListening Object 이벤트가 발생할 때 알림을 받을 객체로써, EventListener 인터페이스를 수행하는 객체이거나, JavaScript function이어야 합니다.

반환

DVE 객체를 반환합니다.

설명

unbind 메소드는 요소가 특정 이벤트가 발생할 때 불려질 리스너를 삭제하고자할 때 사용됩니다.

let svg = new DV().svg(400, 100).into("DV-01");
svg.text("Click the below box", 10, 20);
let shape = svg.rect(10, 50, 200, 20).fill("red");
shape.bind("click", function (evt) {
    alert("I am clicked! Only once!");
    shape.unbind("click", arguments.callee);
    // 다음의 방법도 사용이 가능합니다.
    // evt.target.removeEventListener(evt.type, arguments.callee);
});
