DVE.prototype.trotate()

문법

let ret = object.trotate(angles)

파라미터

이름 형식 설명
angles <list of angle>

텍스트 모든 문자가 개별적으로 회전을 합니다.

angles는 하나의 값(각도)을 갖거나, 콤마 혹은 빈 문자로 구성된 리스트로 나타낼 수 있습니다.

반환

DVE 객체를 반환합니다.

설명

'trotate' 메소드는 렌더링이 되는 'text, tspan' 요소가 렌더링 하는 텍스트를 회전합니다.

리스트 형식을 사용하면 개별 문자의 회전 각도를 지정할 수 있습니다.

<svg width="700" height="100">
  <line x1="0" y1="50" x2="100%" y2="50" stroke="#03A9F4" stroke-width="1"/>
  <text x="0" y="48" font-size="0.65em" fill="#03A9F4">base-line</text>
  <text x="50" y="50" font-size="2em" letter-spacing="10" rotate="90 45 90 25">ABC<tspan dy="20" rotate="45 90">1234567890</tspan> xyz</text>
</svg>

let svg = DV.svg().size(700,100).into("dv-01");
DV.line().point1(0,50).point2("100%",50).stroke("#03A9F4",1).into(svg);
DV.text().txt("base-line").xys(0,48).fsize("0.65em").fill("#03A9F4").into(svg);
let t = DV.text().txt("ABC ").xys(50,50).fsize("2em").attr("letter-spacing",10).trotate([90,45,90,25]).into(svg);
DV.tspan().txt("1234567890").dxys(0,20).trotate([45,90]).into(t);
t.txt(" xyz",true);

다음은 text, tspan 엘리먼터의 rotate 속성값에 따라 문자의 회전이 어떻게 영향 받지에 대한 그림 설명입니다.
