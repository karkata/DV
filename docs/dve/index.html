DVE

DVE 객체는 DV 클래스의 정적 메소드를 통해서 생성되는 객체입니다. DVE 객체는 하나의 SVG 요소를 나타내는 인스턴스 객체이며, SVG 요소의 여러 속성들을 다루기 위한 메소드들을 가지고 있습니다.

A

  • acc: 'accumulate' 속성을 설정합니다.
  • add: 'additive' 속성을 설정합니다.
  • append: 대상 DVE 객체를 메소드를 호출한 DVE 객체의 자식 객체로 등록합니다.
  • arc: 'd' 속성에서 'elliptical arc' 명령어를 설정합니다.
  • arcf: 'd' 속성에서 완전한 원을 그리는 'elliptical arc' 명령어를 설정합니다.
  • aspectRatio: 'preserveAspectRatio' 속성을 설정합니다.
  • attr: DVE 객체의 속성을 반환거나 설정합니다.

B

  • bind: 이벤트를 등록합니다.
  • bound: 'x1', 'y1', 'x2', 'y2' 속성을 설정합니다.

C

  • calcMode: 'calcMode' 속성을 설정합니다.
  • center: 'cx', 'cy' 속성을 설정합니다.
  • clipPathUnits: 'clipPathUnits' 속성을 설정합니다.
  • clipPath: 'clip-path' 속성을 설정합니다.
  • clipRule: 'clip-rule' 속성을 설정합니다.
  • css: 'style' 속성을 설정합니다.
  • csto: 'd' 속성에서 'curveto', 'shorthand/smooth curveto' 명령어를 연속으로 설정합니다.
  • cto: 'd' 속성에서 'curveto', 'shorthand/smooth curveto' 명령어를 설정합니다.

D

  • detach: DOM 구조에서 부모와의 관계를 끊습니다.
  • dim: 'x, y, width, height' 속성을 설정합니다.
  • dxys: 'text, tspan' 요소의 'dx, dy' 속성을 설정합니다.

E

F

  • fill: 'fill, fill-opacity, fill-rule' 속성을 설정합니다.
  • focalPoint: 'fx, fy' 속성을 설정합니다.
  • freeze: 애니메이션 요소에서 'fill' 속성을 'freeze'로 설정합니다.
  • fromToBy: 'from, to, by' 속성을 설정합니다.
  • fsize: 'font-size' 속성을 설정합니다.

G

H

  • hto: 'd' 속성에서 'horizontal lto' 명령어를 설정합니다.

I

  • id: 'id' 속성을 설정합니다.
  • into: 현재 요소를 다른 컨테이너 요소에 삽입합니다.

K

  • keyFrom: 'from' 속성을 설정합니다.
  • keyTo: 'to' 속성을 설정합니다.
  • keyValues: 'values' 속성을 설정합니다.
  • keyTimes: 'keyTimes" 속성을 설정합니다.
  • KeySplines: 'keySplines' 속성을 설정합니다.
  • keyPoints: 'keyPoints' 속성을 설정합니다.

L

  • lto: 'd' 속성에서 'lineto' 명령어를 설정합니다.

M

  • mark: 'marker-start', 'marker-middle', 'marker-end' 속성을 설정합니다.
  • markerDim: 'refX, refY, markerWidth, markerHeight, markerOrient' 속성을 한 번에 설정합니다.
  • markerOrient 'markerOrient' 속성을 설정합니다.
  • markerPoint: 'refX, refY' 속성을 설정합니다.
  • markerSize: 'markerWidth, markerHeight' 속성을 설정합니다.
  • markerUnits: 'markerUnits' 속성을 설정합니다.
  • matrix: 'transform, gradientTransform, patternTransform' 속성에서 'matrix' 명령어를 설정합니다.
  • motionPath: 'path' 속성을 설정합니다.
  • motionRotate: 'rotate' 속성을 설정합니다.
  • mto: 'd' 속성에서 'moveto' 명령어를 설정합니다.

N

  • nofill: 'fill' 속성을 'none'으로 설정합니다.

O

  • oco: 'stop' 요소의 'offset, color, opacity' 속성을 설정합니다.

P

  • pathMethod: 'textPath' 요소의 'method' 속성을 설정합니다.
  • pathSpacing: 'textPath' 요소의 'spacing' 속성을 설정합니다.
  • patternContentUnits: 'patternContentUnits' 속성을 설정합니다.
  • patternUnits: 'patternUnits' 속성을 설정합니다.
  • point: 'x, y' 속성을 설정합니다.
  • point1: 'x1, y1' 속성을 설정합니다.
  • point2: 'x2, y2' 속성을 설정합니다.
  • points: 'points' 속성을 설정합니다.
  • pz: 'd' 요소의 'closepath' 명령어를 설정합니다.

Q

  • qsto: 'd' 요소에서 다중 'quadratic bezier curveto' 명령어를 설정합니다.
  • qto: 'd' 요소에서 'quadratic bezier curveto' 명령어를 설정합니다.

R

  • radius: 'r' 속성을 설정합니다.
  • ref: 'id' 속성 값에 '#' 문자를 붙혀 반환합니다.
  • repeatCount: 'repeatCount' 속성을 설정합니다.
  • repeatDur: 'repeatDur' 속성을 설정합니다.
  • requiredFeatures: 'requiredFeatures' 속성을 설정합니다.
  • resettrans: 모든 변환식을 제거합니다.
  • restart: 'restart' 속성을 설정합니다.
  • rmattr: 요소의 속성을 제거합니다.
  • rotate: 'rotate' 변환식을 설정합니다.
  • round: 'rx, ry' 속성을 설정합니다.

S

  • scale: 'scale' 변환식을 설정합니다.
  • setAutoLayout: 자동 레이아웃 처리를 위한 설정을 합니다.
  • setClass: 'class' 속성을 설정합니다.
  • size: 'width, height' 속성을 설정합니다.
  • skewx: 'skewX' 변환식을 설정합니다.
  • skewy: 'skewY' 변환식을 설정합니다.
  • startOffset: 'startOffset' 속성을 설정합니다.
  • stopColor: 'stop-color' 속성을 설정합니다.
  • stopOffset: 'offset' 속성을 설정합니다.
  • stopOpacity: 'stop-opacity' 속성을 설정합니다.
  • stroke: 'stroke' 속성을 설정합니다.
  • strokeDash: 'stroke-dasharray, stroke-dashoffset' 속성을 설정합니다.
  • strokeLinecap: 'stroke-linecap' 속성을 설정합니다.
  • strokeLinejoin: 'stroke-linejoin' 속성을 설정합니다.
  • strokeOpacity: 'stroke-opacity' 속성을 설정합니다.
  • strokeWidth: 'stroke-width' 속성을 설정합니다.

T

  • tanchor: 'text-anchor' 속성을 설정합니다.
  • time: 'begin, dur, end' 속성을 설정합니다.
  • timeBegin: 'begin' 속성을 설정합니다.
  • timeDur: 'dur' 속성을 설정합니다.
  • timeEnd: 'end' 속성을 설정합니다.
  • timeMax: 'max' 속성을 설정합니다.
  • timeMin: 'min' 속성을 설정합니다.
  • tlength: 'textLength, lengthAdjust' 속성을 설정합니다.
  • toWhat: 'attributeName, attributeType, path' 속성을 설정합니다.
  • translate: 'translate' 변환식을 설정합니다.
  • trotate: 'text, tspan' 요소의 'rotate' 속성을 설정합니다.
  • txt: 'text, tspan, desc, styleBlock' 요소의 내용(innerText)를 설정합니다.

U

  • unbind: 이벤트를 제거합니다.
  • updateLayout: 요소 간의 레이아웃을 갱신합니다.

V

  • viewbox: 'viewBox' 속성을 설정합니다.
  • visible: 'visibility' 속성을 설정합니다.
  • vto: 'd' 속성에서 'vertical lto' 명령어를 설정합니다.

X

  • xhref: 'xlink:href' 속성을 설정합니다.
  • xys: 'text, tspan' 요소의 'x, y' 속성을 설정합니다.

Z

  • zoomAndPan: 'zoomAndPan' 속성을 설정합니다.
