DVE.prototype.rotate()

문법

var ret = object.rotate(angle[, cx, cy]);

파라미터

이름 형식 설명
angle <angle>

좌표축 회전각
cx, cy <coordinate>

회전시킬 좌표

반환

DVE 객체를 반환합니다.

설명

rotate 함수는 SVG 요소의 좌표축을 회전시킬 때 사용합니다.

rotate 함수는 SVG 요소의 transform 속성값에 rotate 명령어를 사용하여 좌표축을 회전시킨 새로운 좌표 시스템을 생성합니다.

rotate(<rotate-angle> [<cx> <cy>]) = translate(<cx>, <cy>) rotate(<rotate-angle>) translate(-<cx>, -<cy>)와 동일합니다.

<svg width="400" height="150">
    <g>
        <line x1="1" y1="1" x2="100" y2="1" stroke="gray" stroke-width="1"/>
        <line x1="1" y1="1" x2="1" y2="50" stroke="gray" stroke-width="1"/>
        <text x="10" y="25">Original coordinate system</text>
    </g>
    <g transform="rotate(30)">
        <line x1="1" y1="1" x2="100" y2="1" stroke="gray" stroke-width="1"/>
        <line x1="1" y1="1" x2="1" y2="50" stroke="gray" stroke-width="1"/>
        <text x="10" y="25">New coordinate system</text>
    </g>
    <g transform="rotate(30, 120, 50)">
        <line x1="1" y1="1" x2="100" y2="1" stroke="gray" stroke-width="1"/>
        <line x1="1" y1="1" x2="1" y2="50" stroke="gray" stroke-width="1"/>
        <text x="10" y="25">New coordinate system</text>
    </g>
    <circle cx="120" cy="50" r="5" fill="red"/>
</svg>

var svg = DV.svg().size(400,150).into("dv-01");
DV.g().into(svg).append(
    DV.line().bound(1,1,100,1).stroke("gray",1),
    DV.line().bound(1,1,1,50).stroke("gray",1),
    DV.text().txt("Original coordinate system").xys(10,25)
);
DV.g().rotate(30).into(svg).append(
    DV.line().bound(1,1,100,1).stroke("gray",1),
    DV.line().bound(1,1,1,50).stroke("gray",1),
    DV.text().txt("New coordinate system").xys(10,25)
);
DV.g().rotate(30,120,50).into(svg).append(
    DV.line().bound(1,1,100,1).stroke("gray",1),
    DV.line().bound(1,1,1,50).stroke("gray",1),
    DV.text().txt("New coordinate system").xys(10,25)
);
DV.circle().center(120,50).radius(5).fill("red").into(svg);
