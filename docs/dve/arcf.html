DVE.prototype.arcf()

문법

let ret = object.arcf(rx[, ry[, xar]]);

파라미터

이름 형식 설명
rx, ry <coordinate>

타원의 모양을 결정짓는 x 축 반지름, y 축 반지름의 크기입니다.

ry가 생략되면 ry는 rx와 같은 값으로 대체됩니다.
xar <degree>

rx, ry의 타원을 원점(0, 0)에 놓여 있다고 할 때, x 축에서 양의 방향으로 주어진 각의 크기만큼 타원을 회전시킵니다.

반환

DVE 객체를 반환합니다.

설명

'arcf' 메소드는 SVG path 요소의 'elliptical arc' 커맨드를 나타냅니다.

'arcf' 메소드가 'arc' 메소드와 다른 점은 현재 위치를 기준으로 하여 아래로, 호가 아닌 완전한 타원을 그립니다.

<svg width="6.5cm" height="3cm" viewBox="0 0 650, 300">
  <path d="M250,50 a100,100 0 1 0 0,200 a100,100 0 1 0 0,-200" fill="none" stroke="red" stroke-width="5"/>
  <path d="M500,50 a100,100 0 1 0 0,200 a100,100 0 1 0 0,-200" fill="none" stroke="green" stroke-width="5"/>
</svg>

let svg = DV.svg().size("6.5cm","3cm").viewbox(0,0,650,300).into("dv-01");
DV.path().fill("none").stroke("red",5).mto(250,50).arc(100,100,0,1,0,0,200,true).arc(100,100,0,1,0,0,-200,true).into(svg);
DV.path().fill("none").stroke("green",5).mto(500,50).arcf(100,100,0).into(svg);
