DVE.prototype.freeze()

문법

var ret = dve.freeze();

파라미터

반환

DVE 객체를 반환합니다.

설명

freeze 메소드는 fill 애니메이션 속성값을 freeze로 설정합니다.

animate, set, animateMotion, animateTransform 요소는 애니메이션이 끝날 때, 대상 요소의 상태를 애니메이션 초기 상태 또는 마지막 상태로 설정할 수 있습니다.

fill 애니메이션 속성이 정해지지 않거나 remove이라면 애니메이션이 끝나고 난 후, 대상 요소는 처음 상태로 돌아갑니다. 반면에 freeze로 설정된다면 대상은 애니메이션의 마지막 상태를 유지하게 됩니다.

<svg width="200" height="100">
    <rect x="25" y="25" width="50" height="50" fill="red">
        <!-- 'fill' 값을 'remove'로 설정하거나 'fill' 속성을 제거합니다. -->
        <animate attributeName="opacity" attributeType="CSS" from="1" to="0" begin="0s" dur="3s" fill="remove"/>
    </rect>
    <rect x="125" y="25" width="50" height="50" fill="red">
        <animate attributeName="opacity" attributeType="CSS" from="1" to="0" begin="0s" dur="3s" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 애니메이션을 수행할 수 있습니다.

var svg = DV.svg().size(200,100).into("dv-01");
DV.rect().dim(25,25,50,50).fill("red").append(
    DV.animate().toWhat("opacity").fromToBy(1,0).time("0s","3s")
).into(svg);
DV.rect().dim(125,25,50,50).fill("red").append(
    DV.animate().toWhat("opacity").fromToBy(1,0).time("0s","3s").freeze()
).into(svg);
