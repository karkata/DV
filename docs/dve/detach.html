DVE.prototype.detach()

문법

let ret = object.detach();

파라미터

반환

DVE 객체를 반혼합니다.

설명

DVE 객체에서 제공되는 함수는 일부 함수를 제외하고 항상 DVE 래퍼 객체를 반환합니다.

이 래퍼 객체는 SVG 엘리먼트를 생성하고 DOM 구조를 구성합니다.

래퍼 객체는 최상의 래퍼객체를 제외하고 대부분 부모 래퍼 객체를 갖습니다.

detach 함수는 현 래퍼 객체와 부모 래퍼 객체와의 관계를 끊고, 각 래퍼가 갖는 SVG 엘리먼트 DOM 관계도 끊습니다.

이 함수는 다른 래퍼 객체로 추가하거나 추가되기 위해서 사용할 수 있습니다.

let svg = DV.svg().size(400,100).into("dv-01");
let button = DV.g().css("cursor","pointer").into(svg);
DV.rect().dim(10,10,100,30).fill("#7c4dff").into(button);
DV.text().txt("Toggle").xys(40,30).fill("white").into(button);

let target = DV.rect().dim(10,50,100,30).fill("#7b1fa2").into(svg);

button.bind("click", function (evt) {
    if (target.parent)
        target.detach();
    else
        target.into(svg);
});
