DVE.prototype.keyTo()

문법

var ret = object.keyTo(value);

파라미터

이름 형식 설명
value <value>

애니메이션 마지막 값을 설정합니다.

마지막 값은 SVG에서만 사용되는 특별한 형식을 제외하고, 표준적으로 SMIL 스펙을 따릅니다.

반환

DVE 객체를 반환합니다.

설명

keyTo 메소드는 animate, set, animateMotion, animateTransform 요소의 to 속성을 설정합니다. to 속성은 애니메이션 대상(attributeName 속성으로 지정됩니다.)의 마지막 값을 의미합니다. value 파라미터는 애니메이션 대상이 되는 속성의 값입니다.

<svg width="400" height="20">
    <rect x="50" y="0" width="10" height="10" fill="red">
        <!-- 박스를 x 좌표 상에서 0부터 390까지 2초동안 이동시킵니다. --> 
        <animate attributeName="x" attributeType="XML" from="0" to="390" begin="0s" dur="2s" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 효과를 보여줄 수 있습니다.

var svg = DV.svg().size(400,20).into("dv-01");
DV.rect().dim(50,0,10,10).fill("red").append(
    DV.animate().toWhat("x").keyFrom(0).keyTo(390).time("0s","2s").freeze()
).into(svg);

KeyValues 메소드를 사용하면, keyFrom, keyTo, keyBy, fromToBy을 사용하여 설정된 속성이 무시됩니다.
