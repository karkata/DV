DVE.prototype.points()

문법

let ret = object.points(listOfPoints);

파라미터

이름 형식 설명
listOfPoints Array of <point>

[a, b] 배열로 나타낸 좌표 형식을 여러개 가지고 있는 목록입니다.

반환

DVE 객체를 반환합니다.

설명

'points' 메소드는 'polygon, polyline' 요소 'points' 속성을 할 때 사용합니다.

<svg width="400px" height="150px">
    <polygon stroke="red" stroke-width="3" points="50,125 100,125 100,100 150,100 150,75 200,75 200,50 250,50 250,25 300,25" fill="none"/>
    <polyline stroke="red" stroke-width="3" points="125,125 175,125 175,100 225,100 225,75 275,75 275,50 325,50 325,25 375,25" fill="none"/>
</svg>

let svg = DV.svg().size(400,150).into("dv-01");
DV.polygon({"points": "50,125 100,125 100,100 150,100 150,75 200,75 200,50 250,50 250,25 300,25"}).stroke("red",3).fill("none").into(svg);
DV.polyline().points([
    [125,125],
    [175,125],
    [175,100],
    [225,100],
    [225,75],
    [275,75],
    [275,50],
    [325,50],
    [325,25],
    [375,25]
]).stroke("red",3).fill("none").into(svg);
