DVE.prototype.pathMethod()

문법

var ret = object.pathMethod(method)

파라미터

이름 형식 설명
method "align"|"stretch"

텍스트가 패스를 따라 렌더링되는 방법을 나타냅니다.

  • align: 글리프의 변형(stretching, warping)없이, 회전, 스케일링, 이동을 사용하여 패스를 따라 렌더링합니다.
  • stretch: 글리프를 모두 패스로 변형하여 렌더링을 하기 때문에, 글리프 자체가 변형될 수 있습니다. 잘 지원하지 않습니다.
  • align 값만 지원합니다.

반환

DVE 객체를 반환합니다.

설명

pathMethod 메소드는 textPath 요소의 method 속성값을 설정할 때 사용합니다. method 속성은 텍스트가 패스를 따라 렌더링할 때, 글리프 변화에 대한 설정을 합니다. 그러나 stretch는 지원하지 않고 align 값만 지원합니다.

<svg width="300" height="200">
    <defs>
        <path id="S01P01" d="M150,50 a50,50 0 0,1 0,100 a50,50 0 0,1 0,-100"/>
    </defs>
    <!-- guide line -->
    <path d="M150,50 a50,50 0 0,1 0,100 a50,50 0 0,1 0,-100" fill="none" stroke="red"/>
    <text>
        <textPath xlink:href="#S01P01" method="stretch">Offset from the start of the ‘path’</textPath>
    </text>
</svg>

var svg = DV.svg().size(300,200).into("dv-01");
DV.defs().into(svg).append(
    DV.path().mto(150,50).arc(50,50,0,0,1,0,100,true).arc(50,50,0,0,1,0,-100,true).id("D01P01")
)
DV.path().mto(150,50).arcf(50,50).nofill().stroke("red").into(svg);
var t = DV.text().into(svg);
DV.textPath().xhref("#D01P01").pathMethod("stretch").txt("Offset from the start of the ‘path’").into(t);
