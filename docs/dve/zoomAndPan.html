DVE.prototype.zoomAndPan()

문법

var ret = object.zoomAndPan(value);

파라미터

이름 형식 설명
value "disable"|"magnify"

disable: 주어진 문서 조각안에서 확대 및 패닝 기능을 사용할 수 없습니다.

magnify: 기본값. 주어진 문서 조각안에서 확대 및 패닝 기능을 사용할 수 있습니다.

반환

DVE 객체를 반환합니다.

설명

zoomAndPan 메소드는 최상위 svg 요소가 확대 및 패닝 기능을 갖도록 할지 설정할 때 사용합니다.

최상위(가장 바깥) svg 요소에만 사용되는 zoomAndPan 속성을 설정하는 메소드입니다. 문서 조각 내부에서는 그래픽 요소들을 확대, 축소하거나 이동하는 기능을 사용할 수 있는데, 최상위 svg 문서 조각에서 zoomAndPan 속성을 disable로 설정하면, 이러한 기능을 사용할 수 없게 됩니다. 기본값은 magnify이므로 기능을 비활성화할 목적이 아니라면 설정할 일이 없습니다.

만약 최상위 svg 요소가 아닌, 내부의 svg 요소에서 호출를 하는 경우에는 설정 값이 아무런 영향을 주지 못합니다.
