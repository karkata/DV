DVE.prototype.strokeDash()

문법

let ret = object.strokeDash(array[, offset])

파라미터

이름 형식 설명
array <dasharray>

테두리에 간격 패턴을 사용합니다.

5,3,2 처럼 콤마 또는 공백 문자로 간격을 설정합니다. 만일 홀수개 값이 주어진다면, 값이 반복됩니다.
offset <percentage>|<length>

간격 패턴이 시작되는 위치를 설정합니다.

퍼센티지 또는 길이 값을 사용할 수 있습니다.

반환

DVE 객체를 반환합니다.

설명

strokeDash 메소드는 SVG 요소 stroke-dasharray, stroke-dashoffset 속성을 설정합니다.

<svg width="200" height="150">
  <path d="M10,30 l100,0" fill="none" stroke="#212121" stroke-width="13" stroke-dasharray="10 5 5 5"/>
  <path d="M10,30 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
  <path d="M10,60 l100,0" fill="none" stroke="#212121" stroke-width="13" stroke-dasharray="10 5"/>
  <path d="M10,60 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
  <path d="M10,90 l100,0" fill="none" stroke="#212121" stroke-width="13" stroke-dasharray="10 5 5 5" stroke-dashoffset="5"/>
  <path d="M10,90 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
  <path d="M10,120 l100,0" fill="none" stroke="#212121" stroke-width="13" stroke-dasharray="10 5" stroke-dashoffset="5"/>
  <path d="M10,120 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
</svg>

let svg = DV.svg().size(200,150).into("dv-01");
