DVE.prototype.csto()

문법

let ret = object.csto(curvepoints[, rel]);

파라미터

이름 형식 설명
curvepoints Array

cubic 베지어 곡선을 그리기 위한 좌표 배열을 목록으로 하는 배열입니다.

하나의 베이저 곡선은 [좌표1, 좌표2, ... 좌표n] 형식이며, 개별 좌표 또한 길이가 2인 배열[x, y]입니다.

즉 curvepoints 파라미터는 베지어 곡선을 목록으로 하는 배열을 의미합니다. [곡선배열1, 곡선배열2, ...]
rel Boolean

true이면, 주어진 위치는 현재 위치의 상대 위치로 인식합니다.

반환

DVE 객체를 반환합니다.

설명

'csto' 메소드는 'path' 요소의 'cto, shorthand/smooth cto' 커맨드를 연속적으로 기술할 때 사용합니다.

'cto, shorhand/smooth cto' 커맨드에 대한 이해는 cto 문서를 참고하시길 바랍니다.

<svg width="5cm" height="4cm" viewBox="0 0 500 400">
  <path fill="none" stroke="red" stroke-width="5" d="M100,200 C100,100 250,100 250,200 S400,300 400,200"/>
</svg>

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-01");
DV.path().fill("none").stroke("red",5)
    .mto(100,200)
    .csto([
        [[100,100],[250,100],[250,200]],
        [[400,300],[400,200]]
    ]).into(svg);
