DVE.prototype.timeMax()

문법

let ret = object.timeMax(value);

파라미터

이름 형식 설명
value <Clock-value>|'media'

애니메이션의 최대 지속 기간을 지정합니다.

max 값은 <Clock-value>, 'media'로 나타낼 수 있습니다.

  • Clock-value: 단순 지속 시간을 나타내는 시간의 길이 값으로써, SVG 문서 조각이 로드된 시점을 기준으로 측정된 타임라인의 위치를 의미합니다. 값은 0 보다 커야합니다.
  • 'media': 내장 미디어 기간으로 단순 지속 시간을 지정합니다. 미디어를 정의하는 요소에만 유효하며, SVG 애니메이션에서는 무시됩니다.

반환

DVE 객체를 반환합니다.

설명

timeMax 메소드는 animate, set, animateMotion, animateTransform 요소의 max 속성을 설정합니다.

'max' 속성은 활동 지속 시간의 최댓값을 설정할 때 사용됩니다.

<svg width="400" height="50">
    <rect x="50" y="0" width="10" height="10" fill="red">
        <animate attributeName="x" attributeType="XML" from="50" to="390" begin="0s" dur="5s" fill="freeze"/>
    </rect>
    <rect x="50" y="30" width="10" height="10" fill="red">
        <animate attributeName="x" attributeType="XML" from="50" to="390" begin="0s" dur="5s" max="3s" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 효과를 보여줄 수 있습니다.

let svg = DV.svg().size(400,50).into("dv-01");
DV.rect().dim(50,0,10,10).fill("red").append(
    DV.animate().toWhat("x").fromToBy(50,390).time("0s","5s").freeze()
).into(svg);
DV.rect().dim(50,30,10,10).fill("red").append(
    DV.animate().toWhat("x").fromToBy(50,390).time("0s","5s").timeMax("3s").freeze()
).into(svg);
