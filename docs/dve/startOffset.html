DVE.prototype.startOffset()

문법

let ret = object.startOffset(offset)

파라미터

이름 형식 설명
startOffset <length>

텍스트가 패스를 따라 렌더링될 때, 시작 위치를 나타내며, 기본값은 0입니다.

만약 퍼센티지 값이라면 패스의 시작은 0%, 패스의 마지막은 100%를 나타냅니다.

반환

DVE 객체를 반환합니다.

설명

startOffset 메소드는 textPath 요소의 startOffset 속성값을 설정할 때 사용합니다. startOffset 속성은 텍스트가 패스를 따라 렌더링될 때, 텍스트가 시작되는 위치를 나타냅니다.

<svg width="300" height="200">
    <defs>
        <path id="S01P01" d="M150,50 a50,50 0 0,1 0,100 a50,50 0 0,1 0,-100"/>
    </defs>
    <!-- guide line -->
    <path d="M150,50 a50,50 0 0,1 0,100 a50,50 0 0,1 0,-100" fill="none" stroke="red"/>
    <text>
        <textPath xlink:href="#S01P01"><tspan dy="-20">Offset from the start of the ‘path’</tspan></textPath>
        <textPath xlink:href="#S01P01" startOffset="50%"><tspan dy="20">Offset from the start of the ‘path’</tspan></textPath>
    </text>
</svg>

let svg = DV.svg().size(300,200).into("dv-01");
DV.defs().into(svg).append(
    DV.path().mto(150,50).arc(50,50,0,0,1,0,100,true).arc(50,50,0,0,1,0,-100,true).id("D01P01")
)
DV.path().mto(150,50).arcf(50,50).nofill().stroke("red").into(svg);
let t = DV.text().into(svg);
DV.textPath().xhref("#D01P01").append(
    DV.tspan().txt("Offset from the start of the ‘path’").dxys(undefined, -20)
).into(t);
DV.textPath().xhref("#D01P01").startOffset("50%").append(
    DV.tspan().txt("Offset from the start of the ‘path’").dxys(undefined, 20)
).into(t);
