DVE.prototype.radius()

문법

let ret = object.radius(r);

파라미터

이름 형식 설명
r <length>

요소의 반지름 크기를 나타냅니다.

반환

DVE 객체를 반환합니다.

설명

radius 메소드는 SVG 요소의 반지름을 설정합니다.

let svg = DV.svg().size(275,150).into("dv-01");
DV.circle().center(75,75).radius(50).stroke("red",5).fill("none").into(svg);
DV.circle().center(200,75).stroke("orange",5).nofill().radius(50).into(svg);
