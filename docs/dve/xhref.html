DVE.prototype.xhref()

문법

var ret = object.xhref(value);

파라미터

이름 형식 설명
value <IRI>

<IRI> 형식으로, 참조되는 객체를 위치를 지정합니다.

반환

dve 객체를 반환합니다.

설명

'xhref' 메소드는 SVG 요소의 'xlink:href' 속성을 설정할 때 사용합니다.

<svg width="200px" height="100px">
    <a xlink:href="https://www.w3.org/TR/SVG11">
        <rect x="15" y="35" width="170" height="30" rx="15" fill="red"/>
        <text text-anchor="middle" x="100" y="55" fill="blue">SVG 1.1</text>
    </a>
</svg>

var svg = DV.svg().size(200,100).into("svg-01");
var a = DV.link().xhref("https://www.w3.org/TR/SVG11").into(svg);
var r = DV.rect().dim(15,35,170,30).round(15).fill("red").into(a);
DV.text().xys(100,55).txt("SVG 1.1").tanchor("middle").fill("white").into(a);
