DVE.prototype.ref()

문법

let ret = object.ref();

파라미터

반환

'#' 문자가 붙은 아이디를 반환합니다.

설명

ref 메소드는 요소의 id 속성값을 반환합니다.

반환되는 요소의 아이디 값은 앞에 '#' 문자가 붙습니다. 이것은 다른 요소에서 IRI 참조를 사용할 수 있도록 하기 위한 목적으로 사용됩니다.

만일 '#' 문자를 포함하지 않는 원래 아이디 값을 얻고 싶다면, id 메소드를 사용하거나 attr 메소드를 사용합니다.

use 요소를 사용될 때, 참조가 사용되며, 이 참조값을 ref 메소드를 사용하여 얻을 수 있습니다.

<svg width="300" height="100">
  <defs>
    <rect id="S01R01" width="50" height="20" fill="blue"/>
  </defs>
  <use xlink:href="#S01R01" x="50" y="25"/>
</svg>

let svg = DV.svg().size(300,100).into("dv-01");
let def = DV.defs().into(svg);
let rect = DV.rect().size(50,20).fill("blue").id("D01R01").into(def);
DV.use().xhref(rect.ref()).point(50,25).into(svg);
// xhref 메소드는 ref 메소드를 사용하지 않고 DVE 객체만 파라미터로 받을 수도 있습니다.
