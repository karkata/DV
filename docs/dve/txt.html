DVE.prototype.txt()

문법

let ret = object.txt(text[, append])

파라미터

이름 형식 설명
text String

렌더링할 텍스트입니다.
append Boolean

현재 요소의 문자 노드를 대체하는 경우 false, 뒤로 추가하는 경우 true입니다.

반환

DVE 객체를 반환합니다.

설명

문자 노드를 갖는 요소의 내용(텍스트)를 변경할 때 사용합니다.

let svg = DV.svg().size(300,150).into("dv-01");

let t1 = DV.text().txt("Original Content - 1").xys(10,50).into(svg);
t1.txt("Replaced Content");

let t2 = DV.text().txt("Original Content - 2").xys(10,100).into(svg);
t2.txt(" Appended Content", true);
