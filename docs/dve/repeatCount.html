DVE.prototype.repeatCount()

문법

var ret = dve.repeatCount(value);

파라미터

이름 형식 설명
value <numeric>|'indefinite'

애니메이션 반복 횟수를 설정합니다.

10진수 부동소수점 형식의 숫자입니다. 이 숫자는 애니메이션을 반복하는 횟수를 의미합니다. 분수식을 사용하면 부분 반복이 가능합니다. 이 값은 반드시 0보다 커야 합니다.

만약 이 값이 indefinite이면, 애니메이션이 무한정 반복되도록 설정됩니다.(문서가 끝날 때까지)

반환

DVE 객체를 반환합니다.

설명

repeatCount 메소드는 animate, set, animateMotion 요소의 repeatCount 속성을 설정합니다. repeatCount 속성은 애니메이션의 반복 횟수를 설정할 때 사용됩니다.

<svg width="400" height="20">
    <rect x="50" y="0" width="10" height="10" fill="red">
        <!-- 박스를 x 좌표 상에서 0부터 390까지 2초동안 이동시킵니다. --> 
        <animate attributeName="x" attributeType="XML" from="0" to="390" begin="0s" dur="2s" repeatCount="2.5" fill="freeze"/>
    </rect>
</svg>

DV를 사용하여 동일한 효과를 보여줄 수 있습니다.

var svg = DV.svg().size(400,20).into("dv-01");
DV.rect().dim(50,0,10,10).fill("red").append(
    DV.animate().toWhat("x").keyFrom(0).keyTo(390).time("0s","2s").repeatCount(2.5).freeze()
).into(svg);
