DVE.prototype.pathSpacing()

문법

var ret = dve.pathSpacing(spacing)

파라미터

이름 형식 설명
spacing "auto"|"exact"

텍스트가 패스를 따라 렌더링할 때, 자간을 계산하는 방법을 나타냅니다.

패스를 따라 텍스트의 위치가 놓여지는 룰에 따라 자간이 정해지는 exact 값만 지원합니다. (auto 미지원)

반환

DVE 객체를 반환합니다.

설명

pathSpacing 메소드는 textPath 요소의 spacing 속성값을 설정할 때 사용합니다. spacing 속성은 텍스트의 자간을 계산하는 방법을 결정하는데, exact 값만 지원(기본)하고 auto는 지원하지 않습니다.

<svg width="300" height="200">
    <defs>
        <path id="S01P01" d="M150,50 a50,50 0 0,1 0,100 a50,50 0 0,1 0,-100"/>
    </defs>
    <!-- guide line -->
    <path d="M150,50 a50,50 0 0,1 0,100 a50,50 0 0,1 0,-100" fill="none" stroke="red"/>
    <text>
        <textPath xlink:href="#S01P01" spacing="auto">Offset from the start of the ‘path’</textPath>
    </text>
</svg>

var svg = DV.svg().size(300,200).into("dv-01");
DV.defs().into(svg).append(
    DV.path().mto(150,50).arc(50,50,0,0,1,0,100,true).arc(50,50,0,0,1,0,-100,true).id("D01P01")
)
DV.path().mto(150,50).arcf(50,50).nofill().stroke("red").into(svg);
var t = DV.text().into(svg);
DV.textPath().xhref("#D01P01").pathSpacing("auto").txt("Offset from the start of the ‘path’").into(t);
