DVE.prototype.visible()

문법

let ret = object.visible(torf);

파라미터

이름 형식 설명
torf 'visible'|'hidden'
  • 'visible': 요소가 화면에 렌더링됩니다.
  • 'hidden': 요소가 화면에 렌더링되지 않습니다. 그러나 요소의 크기만큼 화면을 공간을 차지합니다.

반환

DVE 객체를 반환합니다.

설명

visible 메소드는 화면에 렌더링이 되는 요소에 visibility CSS 속성을 인라인으로 적용할 때 사용합니다.

let svg = new DV().svg(400, 100).into("DV-01");
svg.rect(10, 10, 200, 20).fill("red");
svg.rect(10, 40, 200, 20).fill("green").visible("hidden");
svg.rect(10, 70, 200, 20).fill("blue");
