DVE.prototype.hto()

문법

let ret = object.hto(x[, rel]);

파라미터

이름 형식 설명
x <coordinate>

x 축 상의 위치입니다.
rel Boolean

rel(relative) 값이 true이면, 주어진 위치는 현재 위치의 상대 위치로 인식합니다.

반환

DVE 객체를 반환합니다.

설명

'hto' 메소드는 SVG path 요소의 'horizontal lto' 커맨드를 나타냅니다.

'hto'는 주어진 패스 상에서 x 축과 평행한 직선을 그리는 용도로 사용됩니다.

<svg width="5cm" height="4cm" viewBox="0 0 500 400">
  <path fill="none" stroke="blue" stroke-width="5" d="M100,100 H400 M400,300 H100"/>
</svg>

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-01");
DV.path().d("M100,100 H400 M400,300 H100").nofill().stroke("blue",5).into(svg);

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-02");
DV.path().nofill().stroke("blue",5)
    .mto(100,100).hto(400)
    .mto(400,300).hto(100).into(svg);

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-03");
DV.path().nofill().stroke("blue",5)
    .mto(100,100).hto(300,true)
    .mto(400,300).hto(-300,true).into(svg);
