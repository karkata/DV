DVE.prototype.requiredFeatures()

문법

let ret = object.requiredFeatures(value);

파라미터

이름 형식 설명
value String(list-of-features)|Array(list-of-features)

SVG 특성을 나타내는 문자열 리스트(공백으로 구분된)이나, 특성 문자열을 항목을 갖는 배열입니다.

반환

DVE 객체를 반환합니다.

설명

'switchblock' 메소드를 사용하여 'switch' 컨테이너 요소를 생성하면, 하위 요소들은 'requiredFeatures, requiredExtensions, systemLanguage' 속성을 사용하여 조건에 부합되는 요소만 렌더링을 하도록 구성할 수 있습니다.

requiredFeatures는 SVG 렌더링을 처리하는 장치의 특성을 공백 문자열로 구분하여 나타냅니다.

Examples

<svg width="400px" height="100px">
  <switch>
    <text x="10" y="25" requiredFeatures="http://www.w3.org/TR/SVG11/feature#ColorProfile">A: http://www.w3.org/TR/SVG11/feature#ColorProfile</text>
    <text x="10" y="25" fill="#ccc">B: http://www.w3.org/TR/SVG11/feature#ColorProfile</text>
  </switch>
  <switch>
    <text x="10" y="50" requiredFeatures="http://www.w3.org/TR/SVG11/feature#Filter">A: http://www.w3.org/TR/SVG11/feature#Filter</text>
    <text x="10" y="50" fill="#ccc">B: http://www.w3.org/TR/SVG11/feature#Filter</text>
  </switch>
  <switch>
    <text x="10" y="75" requiredFeatures="http://www.w3.org/TR/SVG11/feature#Script">A: http://www.w3.org/TR/SVG11/feature#Script</text>
    <text x="10" y="75" fill="#ccc">B: http://www.w3.org/TR/SVG11/feature#Script</text>
  </switch>
</svg>

let svg = DV.svg().size(400,100).into("dv-01");
let sw1 = DV.switchblock().into(svg);
DV.text().txt("A: http://www.w3.org/TR/SVG11/feature#ColorProfile").xys(10,25).requiredFeatures("http://www.w3.org/TR/SVG11/feature#ColorProfile").into(sw1);
DV.text().txt("B: http://www.w3.org/TR/SVG11/feature#ColorProfile").xys(10,25).fill("#ccc").into(sw1);
let sw2 = DV.switchblock().into(svg);
DV.text().txt("A: http://www.w3.org/TR/SVG11/feature#Filter").xys(10,50).requiredFeatures("http://www.w3.org/TR/SVG11/feature#Filter").into(sw2);
DV.text().txt("B: http://www.w3.org/TR/SVG11/feature#Filter").xys(10,50).fill("#ccc").into(sw2);
let sw3 = DV.switchblock().into(svg);
DV.text().txt("A: http://www.w3.org/TR/SVG11/feature#Script").xys(10,75).requiredFeatures("http://www.w3.org/TR/SVG11/feature#Script").into(sw3);
DV.text().txt("B: http://www.w3.org/TR/SVG11/feature#Script").xys(10,75).fill("#ccc").into(sw3);
