DVE.prototype.strokeOpacity()

문법

let ret = object.strokeOpacity(opacity)

파라미터

이름 형식 설명
opacity Number

테두리의 불투명도를 설정합니다.

0.0 ~ 1.0까지의 값을 가질 수 있으며, 기본값은 1.0입니다.

반환

DVE 객체를 반환합니다.

설명

strokeOpacity 메소드는 SVG 요소의 stroke-opacity 속성을 설정합니다.

opacity 파라미터는 테두리 불투명도를 설정합니다.

<svg width="200" height="150">
  <path d="M10,30 l100,0" fill="none" stroke="#212121" stroke-width="23" stroke-opacity="1.0"/>
  <path d="M10,30 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
  <path d="M10,60 l100,0" fill="none" stroke="#212121" stroke-width="23" stroke-opacity="0.5"/>
  <path d="M10,60 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
  <path d="M10,90 l100,0" fill="none" stroke="#212121" stroke-width="23" stroke-opacity="0.25"/>
  <path d="M10,90 l100,0" fill="none" stroke="#448AFF" stroke-width="1"/>
</svg>

let svg = DV.svg().size(200,150).into("dv-01");
DV.path().d("M10,30 l100,0").fill("none").stroke("#212121",23,undefined,undefined,undefined,undefined,1.0).into(svg);
DV.path().d("M10,30 l100,0").fill("none").stroke("#448AFF",1).into(svg);
DV.path().d("M10,60 l100,0").fill("none").stroke("#212121",23).attr("stroke-opacity",0.5).into(svg);
DV.path().d("M10,60 l100,0").fill("none").stroke("#448AFF",1).into(svg);
DV.path().d("M10,90 l100,0").fill("none").stroke("#212121",23).strokeOpacity(0.25).into(svg);
DV.path().d("M10,90 l100,0").fill("none").stroke("#448AFF",1).into(svg);
