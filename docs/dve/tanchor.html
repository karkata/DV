DVE.prototype.tanchor()

문법

var ret = object.tanchor(anchor)

파라미터

이름 형식 설명
anchor 'start'|'middle'|'end'

text, tspan과 같은 요소를 사용하여 텍스트를 렌더링할 때, 주어진 위치에서의 텍스트를 정렬하는 방식입니다.

반환

DVE 객체를 반환합니다.

설명

tanchor 메소드는 렌더링이 되는 text, tspan 요소에서 텍스트를 렌더링할 때 주어진 위치에서 텍스트를 정렬하는 방식을 나타내는 text-anchor 속성을 설정하는 용도로 사용됩니다.

<svg width="240" height="120">
    <text x="120" y="20" font-size="2em" fill="#03A9F4" text-anchor="start">start</text>
    <text x="120" y="50" font-size="2em" fill="#03A9F4" text-anchor="middle">middle</text>
    <text x="120" y="80" font-size="2em" fill="#03A9F4" text-anchor="end">end</text>
</svg>

var svg = DV.svg().size(240,120).into("dv-01");
DV.text().txt("start").xys(120,20).fsize("2em").fill("#03A9F4").tanchor("start").into(svg);
DV.text().txt("middle").xys(120,50).fsize("2em").fill("#03A9F4").tanchor("middle").into(svg);
DV.text().txt("end").xys(120,80).fsize("2em").fill("#03A9F4").tanchor("end").into(svg);
