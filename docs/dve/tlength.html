DVE.prototype.tlength()

문법

var ret = object.tlength(length, adjust)

파라미터

이름 형식 설명
length <length>

렌더링되는 텍스트의 길이를 명시적으로 지정합니다.

length로 지정된 길이에 텍스트를 모두 렌더링해야 하기 때문에, kerning, letter-spacing, word-spacing, dx, dy의 값이 모두 새로 조정됩니다.
adjust "spacing"|"spacingAndGlyphs"

length에 지정된 길이에 텍스트를 렌더링할 때, 조정되는 대상을 설정합니다.

  • spacing: 글자 사이의 간격만 조정합니다.(기본값)
  • spacingAndGlyphs: 간격뿐만 아니라, 글자를 렌더링할 때 사용하는 모든 요소가 조정됩니다.

반환

DVE 객체를 반환합니다.

설명

'tlength' 메소드는 렌더링이 되는 'text, tspan, textPath' 요소에서 텍스트가 렌더링되는 길이를 정합니다.

adjust 기본값은 'spacing'으로 글자 사이의 간격만 조정하지만, 'spacingAndGlyphs'로 설정되면, 글자 사이의 간격 뿐만 아니라, 글리프의 모양도 변형됩니다.
