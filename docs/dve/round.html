DVE.prototype.round()

문법

let ret = object.round(rx[, ry]);

파라미터

이름 형식 설명
rx <length>

x 축 방향의 반지름 값입니다.
ry <length>

y 축 방향의 반지름 값입니다.

반환

DVE 객체를 반환합니다.

설명

'round' 메소드는 'rect, ellipse' 요소의 'rx, ry' 속성값을 설정합니다.

두 속성은 x축, y축을 기준으로 하는 반지름 길이를 나타내며, 0 또는 양수의 값을 갖습니다.

<svg width="200" height="60">
    <rect x="10" y="10" width="50" height="20" rx="5" ry="5" fill="red"/>
    <ellipse cx="100" cy="20" rx="20" ry="20" fill="purple"/>
</svg>

let svg = DV.svg().size(200,60).into("dv-01");
DV.rect().dim(10,10,50,20).round(5,5).fill("red").into(svg);
DV.ellipse().center(100,20).round(20,20).fill("purple").into(svg);
