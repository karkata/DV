DVE.prototype.center()

문법

let ret = object.center(cx[, cy]);

파라미터

이름 형식 설명
cx, cy <coordinate>

요소가 렌더링되는 영역의 가운데 위치를 나타내는 좌표(cx, cy)입니다.

반환

DVE 객체를 반환합니다.

설명

center 메소드는 SVG 요소가 화면에 그려질 때, 렌더링되는 바운딩 영역의 가운데 좌표를 설정하는 용도로 사용됩니다.

center 메소드는 'cx, cy' 속성값을 설정하기 때문에, 'circle, ellipse, radicalGradient' 요소들만 사용할 수 있습니다.

let svg = DV.svg().size(275,150).into("dv-01");
DV.circle().center(75,75).radius(50).stroke("red",5).fill("none").into(svg);
DV.circle().radius(50).stroke("orange",5).nofill().center(200,75).into(svg);
