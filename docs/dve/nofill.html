DVE.prototype.nofill()

문법

let ret = object.nofill();

파라미터

반환

DVE 객체를 반환합니다.

설명

그래픽 도형 내부를 fill 속성을 none 값으로 설정할 때 사용하는 메소드입니다.

nofill 메소드는 fill("none") 메소드 호출과 동일합니다.

let svg = DV.svg().size(200,50).into("dv-01");
let g = DV.g().stroke("#b6b6b6", 3).into(svg);
DV.rect().dim(5,5,20,20).fill("none").into(g);
DV.rect().dim(30,5,20,20).nofill().into(g);
