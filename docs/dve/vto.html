DVE.prototype.vto()

문법

var ret = dve.vto(y[, rel]);

파라미터

이름 형식 설명
y <coordinate>

y 축 상의 위치입니다.
rel Boolean

rel(relative) 값이 true이면, 주어진 위치는 현재 위치의 상대 위치로 인식합니다.

반환

DVE 객체를 반환합니다.

설명

'vto' 메소드는 SVG path 요소의 'vertical lto' 커맨드를 나타냅니다.

'vto'는 주어진 패스 상에서 y 축과 평행한 직선을 그리는 용도로 사용됩니다.

<svg width="5cm" height="4cm" viewBox="0 0 500 400">
  <path d="M100,100 V300 M400,300 V100" fill="none" stroke="blue" stroke-width="5"/>
</svg>

var svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-01");
DV.path("M100,100 V300 M400,300 V100").fill("none").stroke("blue",5).into(svg);

var svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-02");
DV.path().fill("none").stroke("blue",5)
    .mto(100,100).vto(300)
    .mto(400,300).vto(100).into(svg);

var svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-03");
DV.path().fill("none").stroke("blue",5)
    .mto(100,100).vto(200,true)
    .mto(400,300).vto(-200,true).into(svg);
