DVE.prototype.qsto()

문법

var ret = object.qsto(curvepoints[, rel]);

파라미터

이름 형식 설명
curvepoints Array

Quadratic cubic 베지어 곡선을 그리기 위한 좌표 배열을 목록으로 하는 배열입니다.

하나의 곡선은 [좌표1, 좌표2, ... 좌표n] 형식이며, 개별 좌표 또한 길이가 2인 배열[x, y]입니다.

즉 curvepoints 파라미터는 Quadratic cubic 베지어 곡선을 목록으로 하는 배열을 의미합니다. [곡선배열1, 곡선배열2, ...]
rel Boolean

rel(relative) 값이 true이면, 주어진 위치는 현재 위치의 상대 위치로 인식합니다.

반환

DVE 객체를 반환합니다.

설명

'qsto' 메소드는 'path' 요소의 'quadratic bezier cto, shorthand/smooth quadratic bezier cto' 커맨드를 연속으로 기술할 때 사용합니다.

'quadratic bezier cto, shorthand/smooth quadratic bezier cto' 커맨드에 대한 이해는 qto 문서를 참고하시길 바랍니다.

<svg width="12cm" height="6cm" viewBox="0 0 1200 600">
  <path d="M200,300 Q400,50 600,300 T1000,300" fill="none" stroke="red" stroke-width="5"/>
</svg>

var svg = DV.svg().size("12cm","6cm").viewbox(0,0,1200,600).into("dv-01");
DV.path().fill("none").stroke("red",5)
    .mto(200,300)
    .qsto([
        [[400,50],[600,300]],
        [[1000,300]]
    ]).into(svg);
