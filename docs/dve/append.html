DVE.prototype.append()

문법

let ret = object.append(arguments);

파라미터

이름 형식 설명
arguments params<DVE>

현재 요소에 추가할 DVE 객체 파라미터들입니다. 콤마 문자를 사용하여 여러 객체를 받을 수 있습니다.

반환

DVE 객체를 반환합니다.

설명

append 메소드는 SVG 요소를 감싼 현재 객체에 다른 DVE 객체들을 추가할 때 사용합니다.

into 메소드와 정반대 역할을 담당하며, into 메소드와 달리 append 메소드는 오로지 DVE 객체만 받습니다.

let svg = DV.svg().size(400,100).into("dv-01");
// svg 요소 하위에 rect 요소가 생성됩니다.
let r = DV.rect().dim(10,10,100,30).fill("#7c4dff").into(svg);
let g = DV.g().into(svg);
// rect 요소를 svg 요소에서 제거한 후, g 요소 하위로 추가합니다. 
g.append(r);
