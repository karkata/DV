DVE.prototype.point()

문법

var ret = object.point(x[, y]);

파라미터

이름 형식 설명
x, y <coordinate>

요소가 렌더링되는 영역의 좌측 상단 위치를 나타내는 좌표(x, y)입니다.

반환

dve 객체를 반환합니다.

설명

'point' 메소드는 SVG 요소 바운딩 영역의 좌측 상단 좌표를 설정합니다.

var svg = DV.svg().size(275,150).into("dv-01");
var rect = DV.rect().dim(10,10,100,50).stroke("red",5).fill("none").into(svg);
// 위치를 변경합니다.
rect.point(50,50);
