DVE.prototype.id()

문법

let ret = object.id(id);

파라미터

이름 형식 설명
id string

요소의 고유 이름입니다.

반환

DVE 객체를 반환합니다.

설명

'id' 메소드는 요소가 갖는 고유 이름을 설정할 때 사용합니다.

파라미터가 주어지면 현재 요소가 반환되고, 만일 주어지지 않는다면 요소에 설정된 고유 이름을 반환합니다.

let svg = new dv().svg(400, 50).into("dv-01");
let rect = svg.rect(10, 10, 200, 20).fill("blue").id("rect1");
alert(rect.id());
