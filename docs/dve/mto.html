DVE.prototype.mto()

문법

let ret = object.mto(x, y[, rel]);

파라미터

이름 형식 설명
x, y <coordinate>

패스상에서 서브 패스를 시작할 좌표입니다.
rel Boolean

서브 패스를 시작하는 좌표로 이동할 때, rel(relative) 값이 true이면, 주어진 좌표(x, y)를 현재 위치의 상대 좌표로 인식합니다.

반환

DVE 객체를 반환합니다.

설명

'mto' 메소드는 'path' 요소의 'mto' 커맨드를 나타냅니다.

'mto'는 주어진 패스 상에서 서브 패스를 시작하기 위한 위치를 설정하는 용도로 사용됩니다.

<svg width="5cm" height="4cm" viewBox="0 0 500 400">
  <path d="M100,100 L400,100 M400,300 L100,300" fill="none" stroke="blue" stroke-width="5"/>
</svg>

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-01");
DV.path().d("M100,100 L400,100 M400,300 L100,300").fill("none").stroke("blue",5).into(svg);

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-02");
DV.path().fill("none").stroke("blue",5)
    .mto(100,100).lto(400,100)
    .mto(400,300).lto(100,300).into(svg);

세번째 파라미터가 true이면 상대좌표로 인식합니다.

let svg = DV.svg().size("5cm","4cm").viewbox(0,0,500,400).into("dv-03");
DV.path().fill("none").stroke("blue",5)
    .mto(100,100).lto(400,100)
    .mto(0,200,true).lto(100,300).into(svg);
